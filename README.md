# Debesh Dinda - Digital CV

This is a **React-based Digital CV / Online Resume** of Debesh Dinda, Full Stack Developer.

## Features
- Fully responsive design with Tailwind CSS
- Includes all sections: Header, Summary, Skills, Education, Internship, Projects, Achievements
- Mobile and desktop friendly
- Ready to be hosted via GitHub Pages

## Technology Stack
- React.js
- Tailwind CSS
- Vite
